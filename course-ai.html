This course provides an introduction to Artificial Intelligence, covering core concepts such as machine learning, data-driven models, and intelligent decision-making systems.

  • Basics of Machine Learning

  • AI applications in real-world systems

Tools & Technologies

  • Python
  • Machine Learning concepts
  • Basic data analysis
  • AI frameworks (overview)

By the end of this course, you will understand the core principles of AI, recognize its applications, and be prepared to explore advanced AI and machine learning technologies.
