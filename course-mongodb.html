Database Technologies

MongoDB

Learn how to work with MongoDB, a powerful NoSQL database, and build scalable, data-driven applications.

Course Overview

This course introduces NoSQL database concepts using MongoDB. You will learn how to store, query, and manage data using a flexible document-based database model.

What You Will Learn

  • NoSQL concepts and MongoDB architecture
  • CRUD operations in MongoDB
  • Working with collections and documents
  • Indexing and aggregation basics
  • Using MongoDB with backend applications

Tools & Technologies

  • MongoDB Server
  • MongoDB Compass
  • Mongo Shell
  • Basic JavaScript

Learning Outcomes

By the end of this course, you will be able to design and manage NoSQL databases, perform efficient queries, and integrate MongoDB with real-world applications.
